# 🎰 Smart-Lotto-Strategy

과거 데이터 통계와 확률 기반의 필터링을 적용한 전략적 로또 번호 생성기입니다.

## 🚀 주요 기능
* **3단계 분석 모드**: 보수, 중간, 공격 모드별 필터 강도 조절
* **5대 핵심 필터**: 합계 범위, 홀짝 비율, 연속수 제한, 저고 비율, 이월수 분석
* **히스토리 관리**: 생성된 번호의 기록 및 모드 저장
* **시각화 UI**: 실제 로또 공 색상을 반영한 직관적인 대시보드

## 🛠️ 사용 기술
* Python 3.x
* Streamlit (Web UI)
* Pandas (Data Analysis)
